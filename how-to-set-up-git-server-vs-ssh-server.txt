git-cервер будем устанавливать на Windows+cygwin. Клиентами будут: Linux, Windows.

Общий порядок действий:
1. Устанавливаем cygwin
2. В cygwin устанавливаем: ssh; git.
3. Настройка ssh-сервера и запуск службы.
4. Проверка сетевого доступа между сервером и клиентскими машинами.
5. На клиентских машинах ставим ssh-client и git.
6. Проверяем сервер на предмет брэндмауэра (не блокирует-ли трафик).
7. Настраиваем ssh-клиента на Windows-host-е.
7. Проверяем работу ssh-сервера с помощью PuTTY с Windows-клиента.
